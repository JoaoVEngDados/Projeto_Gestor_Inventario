<h1>#Projeto: Gestor de Inventário e Previsão de Reabastecimento</h1>
<h2>Neste projeto,  simulei dados de vendas de 500 produtos diferentes usando o NumPy para calcular automaticamente o Estoque de Segurança e o Ponto de Pedido.</h2>
<p>
  O que o programa faz:

  Simulação de Dados: Gera uma matriz de vendas diárias para 500 produtos durante 30 dias.

Análise Estatística: Calcula a média de vendas e a variabilidade (desvio padrão) de cada item.

Cálculo de Supply Chain: Define quanto stock deve existir para cobrir imprevistos (Safety Stock).

Alerta de Compra: Identifica instantaneamente quais produtos precisam de ser comprados hoje usando Máscaras Booleanas.</p>
